## Overview

Shape is a shared foundation for corner treatment across components.

The system defines the categories `small`, `medium`, `large`, and `full`. Each category includes:
- a corner family (`rounded` or `cut`)
- four logical corner sizes

This makes it possible to keep MD2-style defaults while still supporting asymmetric corners and cut-corner shapes.
