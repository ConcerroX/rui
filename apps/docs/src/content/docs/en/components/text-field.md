## Specs {#specs}

Text Field exposes an input-oriented surface and can be grouped with other form controls.

## Guidelines {#guidelines}

Use a text field when the user needs to enter or edit free-form text.

## Implementation {#implementation}

Import the component from `rui/components` and compose it inside semantic form markup.
