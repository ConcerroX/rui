## Usage

Use Surface to visually group related content without introducing button, card, or layout behavior.

### Good fits

- Panel-like content blocks
- Grouped settings sections
- Containers that need a subtle boundary

### Keep layout and spacing separate

Surface is intentionally minimal. Use child wrappers plus `RStack`, `RRow`, `RColumn`, or `RGrid` to handle spacing and layout inside the surface.
