## Usage

Icons are best used as compact visual affordances inside buttons, labels, and supporting UI.

- Let icons inherit color from surrounding content
- Use `label` only when the icon needs to be announced on its own
- Leave decorative icons unlabeled so assistive technology can ignore them
- Prefer simple 24 × 24 SVG artwork for consistent results
